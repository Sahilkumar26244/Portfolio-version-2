### Prerequisites

- Node.js (v14.0.0 or later)
- npm (v6.0.0 or later)

### Installation

1. Clone the repository
   ```bash
   git clone https://github.com/yourusername/resume-portfolio.git
   ```

2. Navigate to the project directory
   ```bash
   cd resume-portfolio
   ```

3. Install dependencies
